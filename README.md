# 3 reel slot machine

Bundled project available in `dist` folder and at [https://nrezvoi.github.io](https://nrezvoi.github.io)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
